Parent surfaces — the boundary. Quiet shadcn-style neutrals, Inter, sentence case, factual copy. No drawn borders, no pigments, no buddy voice; and none of this leaks into the child's room.

```jsx
<ParentSurface title="This week">
  <ParentRow label="Reading days" value="4" />
  <ParentRow label="New words" value="3" />
  <ParentRow label="Drive mode" control={<ParentToggle on onChange={set} />} />
</ParentSurface>
```

Gate entry behind an adult gesture (e.g. two-finger hold) — never a visible drawn-world button.
