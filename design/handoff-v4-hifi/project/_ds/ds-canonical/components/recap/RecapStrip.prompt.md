Recap strip — "Last time…" as three comic panels, one caption each, all spoken aloud by the buddy as they appear.

```jsx
<RecapStrip panels={[
  { art: <BridgeArt />, caption: "Miko found the bridge" },
  { art: <WindArt />,   caption: "the wind was loud" },
  { art: <PawArt />,    caption: "one brave step…" },
]} />
```

Exactly three panels; captions one short clause each.
